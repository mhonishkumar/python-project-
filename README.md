🚀 Built a Digital Clock using Python & Tkinter

Excited to share a simple yet powerful project I recently developed using Python and the Tkinter GUI library — a real-time digital clock application.

🔹 The clock displays the current time in HH:MM:SS AM/PM format
🔹 Updates automatically every second
🔹 Styled with a digital font, black background, and cyan foreground for a modern look
🔹 Implemented using strftime() from the time module
🔹 Used after() method for continuous real-time updates

This project helped me strengthen my understanding of:
✅ GUI development with Tkinter
✅ Event-driven programming
✅ Working with time modules in Python
✅ Real-time UI updates

Small projects like this improve logic building and deepen core programming concepts. Looking forward to building more interactive Python applications!
